PHOTON platform • logger¶

The PHOTON platform • logger is a simple app to create log entries in the reST format, specifically for the ABlog utility for Sphinx.

The app is built in Python using the new Textual UI library.

For more information, visit the docs site for this repo at https://photon-platform.github.io/photon-logger
